## 🔐 **Authentication Required**

When pushing, you'll need to authenticate with GitHub. Choose one:

### **A. GitHub CLI (Simplest)**
```powershell
gh auth login
# Select HTTPS and follow prompts
```

### **B. Personal Access Token (Recommended)**
1. Go to: https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Select scopes: `repo`, `workflow`
4. Generate and copy the token
5. Paste token when prompted for password

### **C. SSH Key (Most Secure)**
```powershell
ssh-keygen -t ed25519 -C "qusai@example.com"
# Add key to GitHub Settings → SSH Keys
git remote set-url origin git@github.com:QusaiALBahri/imageOSINT.git
git push -u origin main
```
